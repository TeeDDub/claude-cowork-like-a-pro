# 주간보고 스킬

주간보고 Word 문서를 만들 때 아래 규칙을 따른다.

## 파일 규칙

- 파일 형식: Word(.docx)
- 파일명: `주간보고_YYYYMMDD_팀명.docx`
  - 날짜는 해당 주 금요일 날짜 기준
  - 날짜가 제공되지 않으면 오늘 날짜 기준 해당 주 금요일로 계산한다
  - 팀명이 제공되지 않으면 파일명에서 팀명을 생략한다

## 보고서 구조

다음 순서로 작성한다.

1. **이번 주 완료 항목** — 항목명, 담당자, 목표 대비 달성률 포함
2. **다음 주 예정 항목** — 항목명, 담당자, 완료 목표일 포함
3. **이슈 및 협조 요청** — 항목명, 현황, 우선순위(높음/중간/낮음), 필요한 협조 내용 포함

## 표 형식

각 섹션은 아래 형식의 표로 구성한다.

| 항목 | 담당자 | 상태/달성률 | 비고 |
|------|--------|-------------|------|

## 서식 규칙

- 분량: A4 1장 이내
- 글꼴: 본문 맑은 고딕 10pt, 섹션 제목 굵게
- 톤: 간결하고 사실 중심, 존댓말 사용 금지
- 숫자: 달성률은 % 표기, 금액은 천 단위 쉼표

## 예외 처리

- 데이터가 없는 항목은 '해당 없음'으로 표기한다. N/A, 공란 사용 금지.
- 이슈가 없으면 이슈 섹션에 '해당 없음'을 적고 표를 생략한다.
- 담당자 정보가 없으면 해당 칸을 '-'로 표기한다.
